1. Нэг компьютерт хоёр үйлдлийн систем суулгаж болох уу?
- Болно. Миний бодлоор үйлдлийн систем бүрээ өөр өөр Hard Disk дээр суулгах шаардлагатай ба аль үйлдлийн системээ Boot хийхээ BIOS-оосоо Hard Disk-ийн сонголтоор шийдэх байх. 
2. Path-д програмаа оруулаагүй бол яах вэ?
- Path-д програмаа оруулаагүй бол ямарваа нэг үйлдлийг гүйцэтгэхэд шаардлагатай програмын зам нь тодорхойгүй буюу уг үйлдлээ гүйцэтгэж чадахгүй болно. Жишээ нь Python код ажиллуулахад PATH-д Python-ий кодийг ажиллуулдаг програм байрлах хэрэгтэй.
3. Фолдерийн нэр нь дундаа зайтай байвал фолдерийг танихад ямар асуудал үүсэх вэ?
- Нэр хоорондын зайг VCS нь коммандын зайтай андуурч, улмаар алдаа заах эрсдэлтэй.
4. Git, Github хоёрын ялгаа юу вэ?
- Github нь програмыг хадгалах үүлэн технологи юм. Git нь Version Control System(VCS) бөгөөд энэ нь файл, програмуудыг зөөх, удирдах зорилготой систем ба хэрэглэгч Git-ээр дамжуулж Github дахь дата бааздаа програмуудаа байршуулах юм.
5. Jupyter notebook IDE мөн үү?
- IDE нь програм бичихэд зориулсан цогц систем бөгөөд үүнд Jupyter notebook багтана.
6. Commit, Push хоёр юугаараа ялгаатай вэ?
- Commit нь Stage хэсэгт байгаа өөрчлөгдсөн файлыг Local Repo-руу байршуулах үйл явц юм. Харин Push нь Local Repo-д байгаа файлаа Remote Repo-руу хуулах явц юм. 
7. Push хийхээс өмнө оло дахин commit хийж болох уу?
- Болно. Task 1-ыг дээрх байдлаар хийж гүйцэтгэсэн. Энэ нь Local Repo-д өөрчлөлтүүдээ бүртгэсээр эцсийн байдлаар Remote Repo-руу мэдээллээ оруулж буйтай ижил.
8. Commit хийхэд Github Repo-д access хэрэгтэй юу?
- Commit хийх нь Local Repo-д өөрчлөлт оруулах тул Github Repo-д access хэрэггүй.
